# Project Structure Document

This document outlines the folder organization for the **hs-app-mobile** project.

## Directory Structure

### lib/
- **core/**: Contains core functionalities and utilities used throughout the application.
- **components/**: Contains reusable UI components.
- **services/**: Contains services for API calls and data fetching.

### backend/
- **controllers/**: Contains controller files to handle requests and business logic.
- **models/**: Contains database models and schemas.
- **routes/**: Contains route definitions.
- **middleware/**: Contains middleware functions.
- **config/**: Contains configuration files for the application.

### docs/
- **api/**: Contains API documentation.
- **user-guides/**: Contains user guides and tutorials.
- **architecture/**: Contains architectural diagrams and decisions.

## Notes
- Ensure proper naming conventions and file organization as the project grows.
